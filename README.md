# Picko

Bu repo artık uygulama arayüzünü `index.html` dosyasından sunar.

## Çalıştırma
- Yerelde: `index.html` dosyasını tarayıcıda açın.

## Not
Daha önce uygulama kodu `README.md` içinde tutulduğu için bazı ortamlarda sayfada `<!DOCTYPE html>` metni düz yazı gibi görünüyordu. Bu yapı kaldırıldı.
